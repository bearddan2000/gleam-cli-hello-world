# gleam-cli-hello-world

## Description
This is a POC project to demonstrate gleam, a functional language like erlang.

## Tech stack
- gleam
- cargo
- rust
- erlang

## Docker stack
- gleamlang/gleam

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
